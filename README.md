### Description
- This is my demo for simple animation *height, new-item*.
- This repo is used for testing typescript/react. It may not work in the future.

### Install

```
- npm install
- npm install -g http-server
- npm install typescript@next
```

### Run

```
- webpack --watch
- http-server
```
